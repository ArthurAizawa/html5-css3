Deixando a letra grande e pequena

Com a tag BIG podemos deixar a letra grande(porém essa tag é absoleto).

Já a tag SMALL deixa a letra pequena.

Texto deletado

Podemos utilizar a tag DEL para deletar um texto, que indica que ele deve ser lido mas não considerado

Texto inserido

Podemos utilizar a tag INS para inserir um texto, deixando-o sublinhado.

Também podemos utilizar a tag U para sublinhar, porém ela é não semântica

Texto sobrescrito

Utilizando a tag SUP podemos colocar um texto para cima, ex: 2*10-6

Texto subescrito

Com a tag SUB, podemos colocar texto abaixo, como o H2O
